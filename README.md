# webapp

## Installing

* `gonew github.com/arunsworld/webapp github.com/arunsworld/kitchensink`
OR
* `gonew github.com/arunsworld/webapp@version github.com/arunsworld/kitchensink`

## Building (frontend)

    cd frontend
    npm install
    ./runme.sh

## Building (backend)

    cd cmd/app
    go run .
